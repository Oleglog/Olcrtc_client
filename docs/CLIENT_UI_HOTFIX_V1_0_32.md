# UI hotfix v1.0.32 — список подключений и настраиваемый фон

Дата: 2026-07-18.

База: `v1.0.31`.

Scope только UI/settings. Datapath, DNS, routing, subscriptions и mobilecore не меняются.

## 1. Главный экран

- Вернуть рабочую структуру `NestedScrollView` до добавления fragment-level particle overlay.
- Все локальные и subscription-профили остаются в одном прокручиваемом списке; второй профиль не должен зависеть от удаления первого.
- Декоративный status mark карточки имеет фиксированную высоту и не участвует в расчёте полной высоты карточки.
- Окно «Добавить подключение» использует `wrap_content` и не растягивает bottom sheet ниже содержимого.

## 2. Частицы

- Fragment-level overlay удаляется; единый overlay живёт в `MainActivity`, поэтому доступен на всех вкладках.
- Два эффекта: снег и дождь.
- Интенсивность: низкая, средняя, высокая.
- Отдельная политика отображения:
  - только при активном VPN;
  - всегда, пока приложение находится на экране.
- Общий переключатель остаётся; при выключении View скрывается и frame callback останавливается.
- Системное отключение анимаций учитывается.
- Новые зависимости не добавляются.

## 3. Настройки

Строка «Частицы на фоне» показывает текущую конфигурацию. Тап открывает редактор:

- включено/выключено;
- снег/дождь;
- низкая/средняя/высокая интенсивность;
- показывать на всех экранах даже без VPN.

Переключатель в строке остаётся быстрым способом включить или выключить сохранённую конфигурацию.

## 4. Приёмка

- При двух и более профилях все карточки доступны прокруткой без удаления соседних профилей.
- Bottom sheet занимает только высоту заголовка и двух действий.
- Снег и дождь визуально различаются; средняя интенсивность заметнее drift из `v1.0.31`.
- В режиме «всегда» эффект виден на Подключении, Подписках, Статистике и Настройках без активного VPN.
- В режиме «при активном VPN» эффект останавливается после disconnect.
- Настройки переживают перезапуск приложения.
- Unit tests, resource linking и signed release APK проходят в GitHub Actions.
- Публикуются arm64-v8a, armeabi-v7a, universal и `SHA256SUMS.txt`.
